# SneakerRatingSystem
CSIT415_03 Group 3 Project

To run, go to XAMPP manager and then to manage servers and click on start all. 
After the servers status is set to running, go to any browser and enter http://localhost/phpmyadmin/ 
Click on import, and then choose the file php123.sql to import. 
Then take all the files and put this on a folder and title it something like SneakerReview.
Take this folder and put this to Applications(if you have Mac), then to XAMPP(has a folder icon), then to htdocs.
Finally, go to your browser and enter http://localhost/SneakerReview (If you name your folder different, then replace SneakerReview)


index.php is homepage and has navigation bar.
view_shoes.php currently lists all the shoes in the database. 
lebron.php is not done and wants to only list this player's shoes
faq.php is a faq
database.php, database_error.php, and php123.sql is to set up and connect the database
